# Scientific and mathematical names

The name borrows a term from physics, chemistry, mathematics, or computing theory, lending the product an air of technical rigor and precision.

## Techniques

- Name after a subatomic particle or fundamental unit for something foundational (Electron, Quark)
- Name after a mathematical variable or structure for a general-purpose or abstract tool (Vector, Scalar, Lambda, Delta)
- Name after a scientific method or framework for a computation-heavy library (TensorFlow, NumPy, PyTorch)
- Pair a scientific root with a plain suffix for a data or infrastructure product (Supabase)

## Examples

TensorFlow, NumPy, PyTorch, Electron, Quark, Lambda, Delta, Vector, Scalar, Supabase

## Pitfalls

- Can misleadingly imply a specific technical architecture the product doesn't actually use.
- These terms already carry precise meanings for technical audiences, so a loose or unrelated use can read as careless.
