# Domain hacks

The web domain's top-level suffix is used as the tail end of the name itself, so the brand and the URL become the same string.

## Techniques

- Split a word across the domain and the TLD so the two together read as a phrase (`del.icio.us`, `bit.ly`, `about.me`)
- Use a country-code TLD as a meaningful word ending rather than a location signal (`youtu.be`, `notion.so`)
- Use a category-signaling TLD as part of the brand's own meaning (`intercom.io`)

## Examples

del.icio.us, bit.ly, youtu.be, about.me, intercom.io, notion.so

## Pitfalls

- The name depends on a specific domain registry or country-code jurisdiction that could change policy or availability.
- Saying the name aloud outside a URL context is awkward, since "dot" has to be spoken.
