# Invented words

Entirely new lexical constructions with no prior dictionary meaning. Because the word starts as a blank slate, it is highly distinctive and fully ownable — every bit of meaning it eventually carries has to be built by the product itself rather than borrowed from an existing word.

## Techniques

- String together phonetically pleasing syllables that form no existing word (Kodak, Xerox)
- Attach a familiar prefix or suffix to an invented stem to make it feel like a real word (Zillow → Zill + ow, Spotify → Spo + tify)
- Favor consonant-vowel patterns that read as language-neutral, so the name travels across markets (Accenture)
- Elongate or stylize a plausible-sounding word until it becomes unique enough to trademark (Häagen-Dazs)

## Examples

Kodak, Xerox, Häagen-Dazs, Accenture, Zillow, Spotify

## Pitfalls

- Conveys zero meaning at launch, so all brand equity must be built from scratch through marketing and repetition.
- Unfamiliar syllable combinations can be hard to pronounce, spell, or recall correctly.
