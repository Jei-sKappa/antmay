# Codename-style names

Names intentionally styled like internal project names, classified missions, or military and intelligence operations, borrowing the intrigue of something not meant for public consumption. They work well when a product wants to feel bigger, more serious, or more secretive than an ordinary product label would suggest.

## Techniques

- `Project + noun` format (Project Apollo, Project Zero)
- Adjective + object pairing evoking a classified program (Longhorn, Blue Book)
- Mythological figure used as a mission name (Gemini)
- Geographic location used as a mission name
- A codename that survives to become the public-facing product name (Copilot)

## Examples

Project Apollo, Manhattan Project, Longhorn, Blue Book, Gemini, Copilot, Project Zero

## Pitfalls

- Codename-style names can sound temporary, like a placeholder that was never replaced.
- Without context, the name communicates intrigue but nothing about the product's actual purpose.
