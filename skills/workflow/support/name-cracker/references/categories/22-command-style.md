# Command-style names

Names that resemble terminal commands, bare verbs, or imperative instructions, so the name itself reads like something you would type or execute. This works especially well for developer tools, where the name doubles as the mental model of the action it performs.

## Techniques

- Bare imperative verb as the whole name (Git, Fetch, Deploy)
- Verb borrowed straight from Unix/CLI vocabulary (Curl, Make)
- Action noun standing in for the product's core motion (Push, Commit, Runway)

## Examples

Git, Curl, Make, Runway, Fetch, Render, Deploy, Commit, Push

## Pitfalls

- Generic commands are hard to search for and even harder to trademark.
- The everyday-verb meaning is common in unrelated contexts, so it can crowd out brand recognition.
